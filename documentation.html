Api Documentation


List of Calls recogniced by this server

/api.v1/RequestLock/RESOURCE
Asks for a Lock on a given RESOURCE
Returns:
  • 200 OK, lock granted
  • 423 Resource already locked

/api.v1/ReleaseLock/RESOURCE/KEY
Returns:
  • 200 OK, lock released
  • 401 Unauthorized, incorrect resource/key
  • 404 Not Found, Lease time expired, lock released

/api.v1/RefreshLeaseTime/RESOURCE/KEY
Returns:
  • 200 OK, Lease time extended
  • 401 Unauthorized, incorrect resource/key
  • 404 Not Found
  • 410 Lease time expired, lock released
  • 423 Resource locked, unable to refresh
